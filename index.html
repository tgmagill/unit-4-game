Gems Collector!

You will be given a random number at the start of the game.

There are four gems below. By clicking on a gem you will add a specific amount of points to your total score.

You win the game by matching your total score to random number, you lose the game if you total score goes above the random number.

The value of each gem is hidden to make it tricky.

Each time the game starts, the game will change the value of each gem.
